# Instalasi Visual Studio Code

Visual Studio Code (VS Code) adalah *code editor* yang ringan namun sangat tangguh. Aplikasi ini akan menjadi perangkat lunak utama untuk berbagai modul praktikum pemrograman dan pengembangan sistem di laboratorium.

Berikut adalah panduan instalasi dan konfigurasi dasarnya.

## Tahap 1: Unduh dan Instalasi

1. Buka situs resmi Visual Studio Code di **[code.visualstudio.com](https://code.visualstudio.com/)**.
2. Klik tombol **Download for Windows** (atau sesuaikan dengan sistem operasi komputer lab).
3. Buka *file* *installer* yang sudah diunduh (biasanya bernama `VSCodeUserSetup-{versi}.exe`).
4. Setujui *License Agreement* dengan memilih **"I accept the agreement"**, lalu klik **Next**.
5. Pada bagian **Select Additional Tasks**, sangat disarankan untuk **mencentang semua opsi** berikut agar navigasi direktori lebih mudah:
   - *Add "Open with Code" action to Windows Explorer file context menu*
   - *Add "Open with Code" action to Windows Explorer directory context menu*
   - *Register Code as an editor for supported file types*
   - **Add to PATH (requires shell restart) — Sangat Penting!**
6. Klik **Next** lalu **Install** dan tunggu hingga proses instalasi selesai.
7. Klik **Finish** untuk langsung membuka VS Code.

## Tahap 2: Instalasi Ekstensi Pendukung

Kekuatan utama VS Code terletak pada ekstensinya. Untuk menunjang jalannya praktikum dan proyek riset di laboratorium, berikut adalah beberapa ekstensi yang wajib atau sangat direkomendasikan untuk diinstal:

### Ekstensi Produktivitas Dasar
* **Prettier - Code formatter:** Membantu merapikan format kode (*indentation*) secara otomatis saat *file* disimpan.
* **Live Server:** Menjalankan *local development server* dengan fitur *live reload* untuk praktikum Web dasar (HTML/CSS/JS).
* **Material Icon Theme:** Mengubah ikon pada struktur *file* dan direktori agar lebih intuitif secara visual.

### Ekstensi Khusus Lingkungan Praktikum
* **Flutter & Dart:** Ekstensi wajib untuk praktikum pengembangan antarmuka dan aplikasi *mobile* lintas platform.
* **Python:** Sangat penting untuk modul *backend*, kecerdasan buatan, atau pengembangan *enterprise* berbasis Python.
* **Database Client (atau SQLTools):** Memudahkan praktikan dalam mengelola koneksi dan mengeksekusi kueri *database* (baik SQL maupun NoSQL) tanpa harus keluar dari *editor*.
* **Solidity:** Opsional, namun sangat berguna jika lab sedang memfasilitasi riset atau eksperimen proyek desentralisasi dan Web3.

## Tahap 3: Cara Memasang Ekstensi

1. Buka aplikasi VS Code.
2. Klik ikon **Extensions** pada bilah menu di sebelah kiri layar (atau gunakan *shortcut* `Ctrl + Shift + X`).
3. Ketikkan nama ekstensi yang diinginkan pada kolom pencarian.
4. Klik tombol **Install** pada ekstensi yang tepat. (Tips: Perhatikan nama pembuat/*author* resminya, seperti "Microsoft" untuk Python atau "Dart Code" untuk Flutter, guna menghindari instalasi ekstensi palsu).
